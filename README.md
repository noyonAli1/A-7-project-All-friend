🌟 KeenKeeper — Nurture Every Friendship

A sleek and modern friendship management web application built to help users strengthen relationships, stay connected, and never lose touch with the people who matter most.

📌 Project Overview

KeenKeeper is a fully responsive React-based web application designed to make friendship management simple and meaningful. It helps users organize their social connections by tracking interactions such as calls, messages, and video chats while providing useful insights into communication habits.

Whether on mobile, tablet, or desktop, the application delivers a smooth and engaging user experience across all devices.

✨ Core Features
👥 Smart Friend Management

Easily browse and manage friends with beautifully designed profile cards that include profile images, friendship tags, online/offline status, and contact details in a responsive layout.

⚡ Activity Timeline

Keep track of every important interaction including Calls, Text Messages, and Video Chats with automatic timestamps displayed in a clean timeline interface.

📊 Friendship Insights

Get a clear visual summary of communication patterns through interactive charts that display the percentage of Calls, Texts, and Video interactions.
